# Secretarios SUD
Sistema de Administracion para secretarios de La Iglesia de Jesucristo de los Santos de los Ultimos Dias (no oficial), desarrollado con PHP, Mysql y LegoBox

- Panel
- Modulo de datos
- Modulo de unidades
- Modulo de secretarios
- Modulo de configuracion

### Plugins Incluidos
- Morris Charts
- jsPDF
- Fullcalendar
- TinyMCE
- Datatables
- Class Upload
